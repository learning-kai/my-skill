# Writing Rules

## Rewrite Checklist
- Keep the main point stable.
- Merge fragments into coherent paragraphs.
- Replace repetitive template phrases with natural transitions.
- Use lists only when they improve clarity.
- Make sentence length and rhythm less uniform.
- End with a real conclusion, not a generic summary line.

## Useful Transformations
- Bullet list -> explanatory paragraph
- Short note fragments -> full sentence flow
- Generic conclusion -> specific takeaway tied to the topic
- Overly formal wording -> clear academic tone with natural cadence

## Phrases to Avoid Repeating
- 在当今时代
- 总的来说
- 值得注意的是
- 综上所述
- 首先/其次/再次/最后 in every paragraph
- 通过以上分析可以看出

## Preferred Structure by Default
- Introduction: define the topic or task context.
- Body: develop the argument in one or more connected paragraphs.
- Closing: give a focused conclusion or recommendation.

## Style Notes
- Use paragraph-first writing.
- Keep the tone formal enough for academic or report writing, but not stiff.
- If the user provides a numbered outline, convert it into prose unless they explicitly want the outline preserved.
